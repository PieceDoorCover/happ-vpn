## Общая информация

**Happ** — кроссплатформенное клиентское приложение, разработанное с использованием **Xray** для работы с пользовательскими сетевыми настройками. Программа доступна на мобильных устройствах и компьютерах и предоставляет инструменты для создания, загрузки, редактирования и хранения профилей подключения в едином интерфейсе.

Приложение позволяет управлять несколькими конфигурациями одновременно, создавать отдельные наборы параметров под различные задачи, сохранять нужные варианты настроек и быстро переключаться между ними. Пользователь может импортировать готовые конфигурационные файлы или самостоятельно создавать новые профили, указывая необходимые параметры вручную.

Благодаря расширенным возможностям настройки Happ позволяет изменять параметры соединения, управлять правилами маршрутизации и контролировать обработку сетевого трафика. Приложение подходит как для простых вариантов конфигурирования, так и для случаев, когда требуется более детальная настройка подключения.

## Основные функции

Приложение предоставляет следующие возможности:

* создание и сохранение профилей подключения;
* загрузка готовых конфигураций;
* ручное добавление параметров соединения;
* изменение существующих настроек;
* настройка правил маршрутизации;
* управление обработкой сетевого трафика;
* локальное хранение пользовательских профилей;
* быстрое переключение между конфигурациями;
* создание отдельных наборов параметров для разных сценариев.

## Хранение данных и защита информации

Все пользовательские профили, файлы конфигураций и параметры подключения сохраняются исключительно на устройстве пользователя.

Приложение не использует внешние облачные хранилища, не выполняет автоматическую синхронизацию настроек между устройствами и не передаёт сохранённые конфигурации на сторонние серверы.

Информация из пользовательских профилей не собирается и не передаётся третьим лицам. Все действия с настройками выполняются локально, поэтому пользователь самостоятельно управляет своими конфигурациями и определяет способ хранения данных.

## Назначение приложения

**Happ** является клиентским инструментом для работы с пользовательскими сетевыми конфигурациями. Приложение предназначено для управления параметрами подключения и не предоставляет собственные услуги сетевого доступа.

Возможности приложения не включают:

* собственные VPN-серверы;
* прокси-инфраструктуру или сервисы передачи данных;
* серверные компоненты обработки сетевого трафика;
* встроенные подписочные механизмы;
* каталоги и хранилища готовых конфигураций.

Для начала использования необходимо самостоятельно добавить параметры подключения. Пользователь может импортировать конфигурационный файл, использовать данные собственного сервера или вручную указать необходимые значения.

Пользователь самостоятельно управляет профилями подключения, выбирает порядок их использования и несёт ответственность за применение приложения в соответствии с законодательством своего региона.
